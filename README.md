# Rock, Paper, Scissors Game

A modern, responsive, web-based implementation of the classic Rock, Paper, Scissors game. Built as a hands-on project to master core frontend development concepts, DOM manipulation, and state management in JavaScript.

---

## 🎮 Features

*   **Dynamic Scoreboards:** Real-time point tracking for both the player and the computer.
*   **First-to-5 Match Condition:** Automatically detects the match winner, disables controls, and prompts a custom game-over screen.
*   **Modern Dark UI:** A polished dashboard style built using native CSS variables, flexbox layout, and subtle micro-interactions.
*   **Clean Visual States:** Instantly reflects active rounds and highlights choices using accessible icons and interactive animations.

---

## 🛠️ Tech Stack

*   **HTML5:** Semantic architecture ensuring solid accessibility and structural flow.
*   **CSS3:** Custom layout styling utilizing modern Flexbox models and centralized design variables for easy theme modification.
*   **JavaScript (ES6+):** Pure vanilla scripting prioritizing structured state management, functional UI synchronization, and efficient event handling.
